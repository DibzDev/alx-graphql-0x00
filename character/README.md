# Character Queries

This directory contains GraphQL queries for fetching character information from the Rick and Morty API.

## Files:
- `character-id-*.graphql`: Queries for specific characters by ID
- `character-id-*-output.json`: API responses for specific character queries
- `characters-page-*.graphql`: Queries for paginated character lists
- `characters-page-*-output.json`: API responses for paginated character queries
