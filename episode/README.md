# Episode Queries

This directory contains GraphQL queries for fetching characters information from the Rick and Morty API.

## Files:
- `episode-page-1.graphql`: Query for characters with ID 1
- `characters-page-1-output.json`: API response for episode ID 1
- `characters-page-2.graphql`: Query for characters with ID 2
- `characters-page-2-output.json`: API response for characters ID 2
- `characters-page-3.graphql`: Query for characters with ID 3
- `characters-page-3-output.json`: API response for characters ID 3
- `characters-page-4.graphql`: Query for characters with ID 4
- `characters-page-4-output.json`: API response for characters ID 4
